Outcome 4

CREATE WEB SOLUTIONS USING A VARIETY OF PROGRAMMING LANGUAGES.


Artifact 1: First complete Website

I learned how to use HTML, CSS and a little bit of JavaScript to create a static (No database) website. This was my favorite project from the first semester. I really enjoyed (and still do) learning new elements and functions and seeing how they make things move around the page. This project set the foundation for everything I learned and will continue to learn about front-end web development.


Artifact 2: Website that creates and retrieves data from a database

I learned how to create and retrieve data to and from a database using PHP. I also learned how to pass variable data to and from post and session arrays. I liked that this assignment allowed me preform full stack functionality. This will help me greatly if I decide to do contracting work for smaller companies how need a full stack website.
